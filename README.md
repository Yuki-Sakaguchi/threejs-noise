# three.jsでシェーダーを使う
* vertexShader
  * 頂点操作用のGSLSコード（頂点シェーダー）
* fragmentShader
  * フラグメント操作用のGLSLコード（フラグメントシェーダー）
* uniform
  * 頂点シェーダーとフラグメントシェーダーで共有される変数リスト
